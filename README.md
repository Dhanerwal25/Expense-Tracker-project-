# Expense-Tracker-project-
Simple Expense Tracker (Python CLI)
A command-line tool built to manage daily expenses using Python’s core data structures.
2. Technical Implementation (The "How")
Dictionaries: "Used to store each expense as a structured record (e.g., {'amount': 500, 'category': 'Food', 'date': '2026-03-10'})."
Lists: "Used as a central database to store multiple expense dictionaries, making it easy to manage the entire collection."
Loops: "Implemented for and while loops to iterate through the expense list, calculate total spending, and provide a continuous user menu."
Robust Input Validation: The application includes logic to handle incorrect user inputs. For example, if a user enters a string/alphabet where a menu number is expected, the system displays an "Invalid Choice" message instead of breaking the flow.
3. Features
Add New Entry: Input amount, date, description and category.
View All Expenses: Clear list of all saved items.
Total Calculation: Automatically sums up all values in the list.
4. Why this approach?
"This project focuses on efficient data handling using Python's built-in collection types without needing external heavy libraries."
